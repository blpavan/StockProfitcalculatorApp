Stock Profit Calculator
Description
The Stock Profit Calculator is a Spring Boot application that calculates stock profit based on a given stock symbol and year. It reads stock price data from CSV files and provides various financial metrics.

Prerequisites
Java JDK 11 or higher
Apache Maven 3.6.0 or higher
An IDE (e.g., IntelliJ IDEA, Eclipse) or a text editor (e.g., VS Code)
